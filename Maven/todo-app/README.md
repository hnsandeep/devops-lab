# Maven To-Do App Demo

This project demonstrates the Maven build lifecycle on Windows with a simple `TodoManager` example.

## Windows commands

Open Command Prompt or PowerShell and run:

1. Navigate to the project:

   cd /d d:\devops-lab\Maven\todo-app

2. Validate the project:

   mvn validate

3. Compile the application:

   mvn compile

4. Run tests:

   mvn test

5. Package the application:

   mvn package

6. Install to the local repository:

   mvn install

7. Clean build artifacts:

   mvn clean

## Files created

- `pom.xml`
- `src/main/java/com/example/TodoManager.java`
- `src/test/java/com/example/TodoManagerTest.java`
